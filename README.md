# Game Scoreboard Management System

## Project Description
This project is a Java-based Game Scoreboard Management System developed using Object-Oriented Programming concepts. It allows users to manage players and their scores efficiently.

## Features
- Add new players
- Update player scores
- Display leaderboard in descending order
- Reset all player scores

## Technologies Used
- Java
- ArrayList
- Collections Framework
- Comparator

## Files
- `Player.java` - Handles player details and score management
- `Scoreboard.java` - Manages players and leaderboard operations
- `Main.java` - Driver class to execute the application
